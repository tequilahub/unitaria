## Contributing to `unitaria`

**If you found a bug**

Check if the bug was already reported, by searching for it under [Issues](https://github.com/tequilahub/unitaria/issues), and create a [new issue](https://github.com/tequilahub/unitaria/issues/new) if it was not. Often, a code example to reproduce the bug helps in resolving it quicker.

**If you want to fix a bug**

Fork the repository, optionally create a new branch (with `git checkout <name-of-new-branch>`), implement the fixes and open a [new pull request](https://github.com/tequilahub/unitaria/pulls/new). If there is an open issue for the bug, it is helpful to let people there know that you are working on the bug. In this case, please also not the issue number in the pull request.

**If you are looking for a new feature**

Check if the feature was already requested, by searching for it under [Issues](https://github.com/tequilahub/unitaria/issues), and create a [new feautre request issue](https://github.com/tequilahub/unitaria/issues/new) if it was not.
You are welcome to start working on the new feature yourself immediately, but please note that we might decide against including the feature.

**If you have any other questions**
Please share it by creating a new [discussion](https://github.com/tequilahub/unitaria/discussions). We will try to get back to you as quick as possible. Feedback helps to make `unitaria` better for everyone.
